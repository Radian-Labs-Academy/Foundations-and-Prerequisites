# Radian Labs — Foundations Reference Code

This is the **official reference repository** for the Radian Labs AMR program. Every piece of
code you copy during a lesson lives here, organized to mirror the workspace you build on your
own machine. You clone it once and copy from it as each lesson tells you to.

> You do **not** run your coursework from inside this repo. You build your own workspace at
> `~/radian_ws` and copy reference files into it. This repo is the *source of truth* you copy
> *from*.

---

## The Radian Labs Workspace Convention

Every lesson and every future course in the program uses one workspace, on your machine, at:

```
~/radian_ws/
├── practice/          # hand-written drills (Linux, early lessons)
│   └── module-01/
│       └── lesson-02/
├── src/               # ROS2 packages (from Lesson 1.10 onward)
│   └── radian_intro/
├── build/  install/  log/    # generated by colcon — never edit by hand
└── README.md
```

- **`practice/`** — where you do the hands-on Linux drills in Module 01. One folder per lesson.
- **`src/`** — where ROS2 packages go once you reach the build-system lesson. All Radian Labs
  packages are named `radian_*` (e.g. `radian_intro`, `radian_bringup`).
- **`build/ install/ log/`** — created automatically by `colcon`. You never touch these.

This repo mirrors that exact layout (`practice/` and `src/`) so a file's place here tells you
exactly where it goes in your workspace.

## How To Use This Repo

```bash
# Clone it once, into your home directory:
cd ~
git clone https://github.com/<your-radian-labs-org>/radian-labs-foundations.git

# Each lesson then tells you which file to copy into your workspace, e.g.:
cp ~/radian-labs-foundations/practice/module-01-linux/lesson-02-terminal/hello_radian.sh \
   ~/radian_ws/practice/module-01/lesson-02/
```

Pull the latest reference code any time with:

```bash
cd ~/radian-labs-foundations && git pull
```
