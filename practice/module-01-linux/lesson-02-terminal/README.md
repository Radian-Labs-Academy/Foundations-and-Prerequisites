### The workflow this lesson teaches

1. **Terminal 1 (Workspace)** — build the organized folder structure under `~/radian_ws`.
2. **Terminal 2 (Editor)** — create the file and copy this reference code into it.
3. **Terminal 3 (Runner)** — run the script and read its output.
4. **Infer** — confirm what the output proves about your setup.
